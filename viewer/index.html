Point the viewer at an FileHook JSONL file from a recent agent run.

from agentx_dev import observability, FileHook, config
config.observability_enabled = True
observability.add_hook(FileHook("./trace.jsonl"))

# ... run your agent ...

Then drag trace.jsonl here or use the button above.
